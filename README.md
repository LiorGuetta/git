# Sudoku Solver and Generator (C)

A modular C application that generates valid Sudoku puzzles and solves them using a recursive backtracking algorithm.

## Project Structure
- src/: Implementation files.
    - main.c: Handles program flow and user output.
    - sudoku.c: Contains the core generation and solving logic.
- inc/: Header files.
    - sudoku.h: Function prototypes and definitions.
- Makefile: Build automation script.

## Getting Started

### Prerequisites
- A C compiler (GCC recommended)
- make build utility

### Building the Project
To compile the project and create the executable, run:
```bash
make
